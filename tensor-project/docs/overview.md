# Tensor Project Overview

This repository is organized as a practical learning-and-building project.

## Modules

### `tensors.py`
Core tensor operations:
- unfolding
- folding
- mode-n products
- Frobenius norm
- rank-1 outer products

### `decompositions.py`
Linear algebra and tensor approximation:
- truncated SVD
- matrix rank from singular values
- best rank-k approximation
- CP-style rank-1 tensor approximation

### `pca.py`
Principal component analysis:
- centering
- covariance computation
- dimensionality reduction
- explained variance ratios

### `fourier.py`
Frequency-domain tools:
- FFT spectrum
- dominant frequency estimation
- inverse reconstruction
